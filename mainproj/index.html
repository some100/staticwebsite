Games currently active: SM64, Sonic CD (save non-functional), Sonic Mania (save non-functional) and Zelda 3

Note: For Sonic Mania, due to the high file size of the RSDK data file required for the game to be playable, it'll take somewhat long of a time to get to the splash screen depending on your download speed. Another thing to note is that it requires at least 2 cores, 1 GB RAM, and the game speed is also dependent on refresh rate. The baseline refresh rate is 60 hz, and having a refresh rate such as 75 hz would cause the game to run 1.25x as fast, 45 hz would cause the game to run 0.75x as fast, and so on.

Controls for SM64

WASD - Movement
L - Jump (A)
, - Punch/Grab (B)
K - Crouch (Z)
Arrow Keys - Camera Movement (C Stick)
Spacebar - Start
Right Shift - Switch Camera Perspectives (R)

Controls for Sonic 1 / 2

WASD - Movement
L - Jump and Select (A)
; - Pause and Back (B)
Enter - Start
Mouse 1 - Select / Start game (Sonic 2)

Controls for Sonic CD

WASD - Movement
L - Jump and Select (A) ; - Jump and Back (B) ' - Jump and Select (C)
Enter - Start and Main Menu

Controls for Sonic Mania

Up, Down, Left, Right arrow - Movement
A - Jump and Select
S - Jump and Back
Q - Jump and Delete
W - Auxiliary
Enter - Start and Main Menu

Controls for Zelda3

WASD - Movement
P - A
L - B
O - X
K - Y
Q - L
E - R
Enter - Start
Tab - Select
Note: These controls map to the SNES controller, since that was where this game originally was from.
